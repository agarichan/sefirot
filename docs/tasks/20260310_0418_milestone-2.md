```python
# src/toy/fizzbuzz.py

def fizzbuzz(n: int) -> list[str]:
    """1 から n までの FizzBuzz 結果をリストで返す。

    - 3 の倍数 → "Fizz"
    - 5 の倍数 → "Buzz"
    - 3 と 5 の両方の倍数 → "FizzBuzz"
    - それ以外 → 数字の文字列（例: "1", "2"）

    Args:
        n: 正の整数

    Returns:
        長さ n のリスト
    """
```

#### import 先（既存コード）
なし。標準ライブラリのみ使用。

#### 実装パターン
- シンプルなループ処理。特別なパターンなし。

#### テスト仕様

```python
# tests/toy/test_fizzbuzz.py
import pytest
from src.toy.fizzbuzz import fizzbuzz

# 正常系
# - fizzbuzz(15) の結果が正しいこと（"1", "2", "Fizz", "4", "Buzz", ..., "FizzBuzz"）
# - fizzbuzz(1) → ["1"]
# - fizzbuzz(3) → ["1", "2", "Fizz"]
# - fizzbuzz(5) の5番目が "Buzz" であること

# 境界値
# - fizzbuzz(0) → []（空リスト）

# 異常系
# - fizzbuzz(-1) → ValueError
```

#### 注意事項
- `n=0` は空リストを返す
- `n < 0` は `ValueError` を raise する

### ステップ B: impl-fibonacci（Wave 1）

#### 作成するファイル
- `src/toy/fibonacci.py`
- `tests/toy/test_fibonacci.py`

#### 型定義・インターフェース

```python
# src/toy/fibonacci.py

def fibonacci(n: int) -> list[int]:
    """先頭 n 個のフィボナッチ数列を返す。

    0-indexed: [0, 1, 1, 2, 3, 5, 8, ...]

    Args:
        n: 取得する個数（0以上の整数）

    Returns:
        フィボナッチ数列のリスト
    """

def fib(n: int) -> int:
    """n 番目のフィボナッチ数を返す（0-indexed）。

    fib(0) = 0, fib(1) = 1, fib(2) = 1, fib(3) = 2, ...

    Args:
        n: 0以上の整数

    Returns:
        n 番目のフィボナッチ数
    """
```

#### import 先（既存コード）
なし。標準ライブラリのみ使用。

#### 実装パターン
- `fibonacci` はイテレーティブに計算してリストに追加するパターン
- `fib` は `fibonacci` を呼ぶか、独立して計算してもよい

#### テスト仕様

```python
# tests/toy/test_fibonacci.py
import pytest
from src.toy.fibonacci import fibonacci, fib

# 正常系
# - fibonacci(7) → [0, 1, 1, 2, 3, 5, 8]
# - fib(0) → 0
# - fib(1) → 1
# - fib(10) → 55

# 境界値
# - fibonacci(0) → []
# - fibonacci(1) → [0]
# - fibonacci(2) → [0, 1]

# 異常系
# - fibonacci(-1) → ValueError
# - fib(-1) → ValueError
```

#### 注意事項
- `n=0` のとき `fibonacci` は空リスト、`fib` は `ValueError`（0番目は存在するので fib(0)=0 が正しい。負の数のみ ValueError）
- 訂正: `fib` で `n < 0` のときのみ `ValueError`

### ステップ E: impl-caesar（Wave 1）

#### 作成するファイル
- `src/toy/caesar.py`
- `tests/toy/test_caesar.py`

#### 型定義・インターフェース

```python
# src/toy/caesar.py

def encrypt(text: str, shift: int) -> str:
    """シーザー暗号で暗号化する。

    英字（a-z, A-Z）のみシフトし、他の文字はそのまま。
    大文字は大文字のまま、小文字は小文字のまま。

    Args:
        text: 暗号化する文字列
        shift: シフト量（正: 右シフト、負: 左シフト）

    Returns:
        暗号化された文字列
    """

def decrypt(text: str, shift: int) -> str:
    """シーザー暗号を復号する。

    encrypt(text, shift) の逆操作。

    Args:
        text: 復号する文字列
        shift: encrypt 時に使用したシフト量

    Returns:
        復号された文字列
    """
```

#### import 先（既存コード）
なし。標準ライブラリのみ使用。

#### 実装パターン
- `decrypt` は `encrypt(text, -shift)` で実装可能
- `ord()` と `chr()` を使って文字コードで計算する

#### テスト仕様

```python
# tests/toy/test_caesar.py
import pytest
from src.toy.caesar import encrypt, decrypt

# 正常系
# - encrypt("abc", 3) → "def"
# - encrypt("xyz", 3) → "abc"（ラップアラウンド）
# - encrypt("Hello, World!", 5) → "Mjqqt, Btwqi!"
# - decrypt("def", 3) → "abc"
# - decrypt(encrypt("test", 13), 13) → "test"（ラウンドトリップ）

# 境界値
# - encrypt("", 5) → ""（空文字列）
# - encrypt("abc", 0) → "abc"（シフト0）
# - encrypt("abc", 26) → "abc"（26シフト = 元に戻る）
# - encrypt("abc", -1) → "zab"（負のシフト）

# 異常系
# - 特になし（全ての文字列・整数の組み合わせが有効）
```

#### 注意事項
- シフト量が 26 を超える場合も正しく動作すること（mod 26）
- 負のシフトも正しく動作すること

### ステップ G: impl-reverse（Wave 1）

#### 作成するファイル
- `src/toy/reverse.py`
- `tests/toy/test_reverse.py`

#### 型定義・インターフェース

```python
# src/toy/reverse.py

def reverse_string(s: str) -> str:
    """文字列を反転する。

    Args:
        s: 反転する文字列

    Returns:
        反転された文字列
    """

def reverse_words(s: str) -> str:
    """単語の順序を反転する。

    単語は空白で区切られる。連続する空白は単一の空白として扱う。

    例: "hello world" → "world hello"

    Args:
        s: 反転する文字列

    Returns:
        単語順序が反転された文字列
    """
```

#### import 先（既存コード）
なし。標準ライブラリのみ使用。

#### 実装パターン
- `reverse_string`: スライス `s[::-1]` で実装
- `reverse_words`: `split()` して `reverse` して `join` するパターン

#### テスト仕様

```python
# tests/toy/test_reverse.py
import pytest
from src.toy.reverse import reverse_string, reverse_words

# 正常系
# - reverse_string("hello") → "olleh"
# - reverse_string("abcde") → "edcba"
# - reverse_words("hello world") → "world hello"
# - reverse_words("the quick brown fox") → "fox brown quick the"

# 境界値
# - reverse_string("") → ""
# - reverse_string("a") → "a"
# - reverse_words("") → ""
# - reverse_words("hello") → "hello"（単語1つ）
# - reverse_words("  hello  world  ") → "world hello"（余分な空白）

# 異常系
# - 特になし
```

#### 注意事項
- `reverse_words` は `str.split()`（引数なし）を使えば連続空白・先頭末尾の空白を自動処理できる

```python
def binary_search(arr: list[int], target: int) -> int:
    """ソート済みリストから target のインデックスを返す。

    見つからなければ -1 を返す。

    Args:
        arr: ソート済みの整数リスト
        target: 検索する値

    Returns:
        target のインデックス。見つからなければ -1。
    """
```

#### import 先（既存コード）
なし。標準ライブラリのみ使用。

#### 実装パターン
- 標準的な二分探索（while ループ、left/right ポインタ）

#### テスト仕様

```python
# tests/toy/test_binary_search.py
import pytest
from src.toy.binary_search import binary_search

# 正常系
# - binary_search([1, 3, 5, 7, 9], 5) → 2
# - binary_search([1, 3, 5, 7, 9], 1) → 0（先頭）
# - binary_search([1, 3, 5, 7, 9], 9) → 4（末尾）
# - binary_search([1, 3, 5, 7, 9], 4) → -1（存在しない）

# 境界値
# - binary_search([], 5) → -1（空リスト）
# - binary_search([1], 1) → 0（要素1つ・存在する）
# - binary_search([1], 2) → -1（要素1つ・存在しない）
# - binary_search([1, 2], 1) → 0（要素2つ・先頭）
# - binary_search([1, 2], 2) → 1（要素2つ・末尾）

# 異常系
# - 特になし（ソート済みリストを前提とする）
```

#### 注意事項
- 重複要素がある場合、いずれかのインデックスを返せばよい（特定の要素を返す必要はない）
- `bisect` モジュールは使わずに自前実装すること
